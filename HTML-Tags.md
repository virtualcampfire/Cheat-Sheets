# HTML-Tags#

````
<!DOCTYPE html>:         Deklaration des Dokumententyps.
<html>:                  Start des HTML-Dokuments.
<head>:                  Enthält Meta-Informationen über die Webseite, z.B. den Seitentitel und Metadaten.
<title>:                 Setzt den Titel der Webseite, der im Browser-Tab angezeigt wird.
<meta>:                  Definiert Metadaten wie Zeichencodierung und Beschreibung.
<body>:                  Der Hauptinhalt der Webseite wird hier platziert.
<h1>, <h2>, <h3>, ...:   Überschriften erster bis sechster Ebene.
<p>:                     Absatz.
<a>:                     Hyperlink.
<img>:                   Einbinden eines Bildes.
<ul>:                    Ungeordnete Liste.
<ol>:                    Geordnete Liste.
<li>:                    Listenelement innerhalb von <ul> oder <ol>.
<table>:                 Erstellt eine Tabelle.
<tr>:                    Zeile in einer Tabelle.
<th>:                    Tabellenüberschrift.
<td>:                    Tabellenzelle.
<form>:                  Formularelement zur Dateneingabe.
<input>:                 Eingabefeld innerhalb eines Formulars.
<label>:                 Beschriftung für Eingabefelder.
<button>:                Schaltfläche innerhalb eines Formulars.
<div>:                   Container zur Gruppierung von Elementen für CSS-Styling und Layout.

```
